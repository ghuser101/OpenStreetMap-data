Explanation of project files :
OpenStreetMapDataCaseStudy.pdf - PDF file containing references, link to map position as well as explanation of data wrangling and answers to project questions.
shape_element_main_code.py - Main project code.
uploadcsv.py - contains code to upload csv files and insert into SQL db.
sample.osm - sample of the data on selected map area.
schema.py - contains schema file for validation purposes.
audit_postcode.py - Audit function for postal codes
audit_street_names.py - audit function for street names
clean_streetnames.py - cleaning function for street names.
clean_postcode.py - cleaning function for postal codes.
